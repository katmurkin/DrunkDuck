GG7404 GEOWEB PROJECT


GETTING STARTED

This webmap seeks to provide information relating to points of interest along the canal network in the wider Birmimgham area. 


PREREQUISITES

None


INSTALLATION

Copy all files in zip folder into a suitable location on web server.  Navigate to Map.html file using web server.  Note that webpage will not load correctly if loaded from the local filesystem (file:/// link).  All links are relative to Map.html file.  canalData.json and pub_Database.json must remain in the GeoJSON subdirectory

JQuery, Leaflet and Turf libraries are accessed via the internet and do not require any installation.

To uninstall, simply delete the files
